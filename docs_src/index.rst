sbipf-reporter
==============

SBI証券のポートフォリオCSVから、わかりやすいレポートを生成するCLIツール。

**対応フォーマット:**
  - 端末表示（rich テーブル）
  - Markdown ファイル出力
  - CSV ファイル出力（損益率付き）
